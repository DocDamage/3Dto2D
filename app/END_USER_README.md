# Start here

Launch SpriteForge Studio v12 from the package root with:

```text
START_HERE.bat
```

The browser dashboard handles setup, ComfyUI launch, WAN generation, video conversion, QA, repair, project sharing, queue monitoring, and engine export.

For the full workflow, open `docs/END_USER_GUIDE.md`. For licensing and external payload details, see `LICENSE` and `THIRD_PARTY_NOTICES.md` in the package root.
