Guided workflow
Launchpad

      

        
01
Prove the app works
Run the no-GPU demo first. If this fails, fix SpriteForge before touching WAN.

        
02
Finish setup
Install ComfyUI, nodes, Manager, and the safe Wan 2.1 model set. Then launch ComfyUI.

        
03
Generate the first sprite
Use debug first. Once that works, switch to the RTX 3060 / Wan 2.1 or Wan 2.2 5B tier.

        
04
QA and repair
Click a recent sprite, run QA, auto-fix jitter/loop issues, then preview the result.

        
05
Build a release
Package finished sprites, metadata, preview GIFs, engine notes, and a preflight report into a release ZIP.

        
06
Batch a character
Create a persistent production queue for idle, walk, run, attacks, hurt, and directions.
