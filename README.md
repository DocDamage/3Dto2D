# SpriteForge Studio v12 — Auto WAN Edition

Welcome to **SpriteForge Studio**, a professional, local pipeline to generate game-ready transparent, looping 2D sprite sheets directly from text prompts, reference images, or existing video files. 

By wrapping ComfyUI and WAN (Hunyuan Latent Video) models into an operational desktop dashboard, SpriteForge automates video rendering, background removal (chroma keying), anchor stabilization, loop-seam stitching, and engine-ready exports.

## 🚀 Quickest Launch Path

To start SpriteForge Studio, simply run the launcher script from the root directory:

1. Double-click **`START_HERE.bat`** (or execute `app/spriteforge_launcher.py` in your terminal).
2. The local web server will launch and automatically open the SpriteForge Dashboard in your web browser.
3. The dashboard will guide you through active projects, live commands, and system statuses.

## 💻 GPU vs. No-GPU Paths

SpriteForge Studio adapts automatically to your hardware configuration.

### 🟢 1. Local GPU Mode (RTX 3060+ with 12GB+ VRAM)
To generate completely new sprites locally:
1. Make sure you have **Git** and **Python 3.10+** installed.
2. In the **Setup** tab of the dashboard, click **Install Everything: Safe Wan 2.1**. This sets up ComfyUI, custom nodes, and downloads the default 9.8 GB Wan 2.1 T2V 1.3B models.
3. Launch ComfyUI via **Launch ComfyUI** (or run `START_HERE.bat --comfy`).
4. Generate sprites under the **Make Sprite** tab.

### 🟡 2. No-GPU Mode / CPU Mode (Demo & Video Conversion)
You don't need a high-end GPU to run SpriteForge's processing, packing, or QA tools:
- **Run the Demo:** Double-click **`RUN_DEMO_NO_GPU.bat`**. This runs a test conversion on a prebuilt video, validating that pillow, ffmpeg, and chroma-key utilities are functional on your CPU.
- **Convert Existing Videos:** If you generated a video elsewhere (e.g., on a cloud GPU, RunPod, or a remote ComfyUI server), drop the `.mp4`/`.webm` in the **Convert Video** tab to remove the background and slice it into a sheet locally.

| :--- | :--- |
