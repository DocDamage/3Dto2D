SpriteForge Studio v12 - Auto WAN Installer

Recommended first install:
1. Double-click INSTALL_EVERYTHING_WITH_WAN.bat.
2. It installs SpriteForge deps, ComfyUI, WAN nodes, ComfyUI Manager, and automatically downloads the Wan 2.1 T2V 1.3B model files.
3. After it finishes, double-click START_HERE.bat to open the visual UI.

The model download is large: roughly 9.8 GB plus temporary cache space. If it is interrupted, run INSTALL_EVERYTHING_WITH_WAN.bat again; it resumes/skips completed files.
